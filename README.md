# EPA App – Mobile Frontend

This repository contains the mobile frontend for the EPA App, built with React Native and Expo.

![EPA App Preview](./assets/images/epa3.webp)

---

## Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/tlxq/epa-appen.git
   cd epa-appen/mobile
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

   or

   ```bash
   yarn install
   ```

3. **Start Expo**

   ```bash
   npx expo start
   ```

   - You can run on iOS simulator, Android emulator, or with the Expo Go app on your device.
   - Make sure your device is on the same Wi-Fi network as your computer if you test with a physical phone.

---

## How it works

- The app connects to a backend API at [`https://api.ttdevs.com`](https://api.ttdevs.com) by default.
- No environment file setup is required—everything works out of the box with the default backend URL.
- You can log in with provided test credentials or register via an invite (contact project admin if needed).

---

## Troubleshooting

If you experience network errors:

- Make sure [`https://api.ttdevs.com`](https://api.ttdevs.com) is live and reachable.
- Restart the Expo app if needed.
